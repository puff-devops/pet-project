# Pet DevOps Project

Этот проект демонстрирует полный CI/CD pipeline:

## Технологии:
- Jenkins для CI/CD
- Nexus для хранения артефактов
- Docker для контейнеризации
- Kubernetes (Minikube) для оркестрации
- Grafana + Prometheus для мониторинга

## Как запустить:
1. Клонировать репозиторий
2. Запустить docker-compose up
